# Finance Tracker

A React-based personal finance tracker for logging income and expenses, categorizing transactions, and visualizing balances in real time.

## About This Project

This project is part of my hands-on exploration of **Claude Code**, Anthropic's agentic coding assistant. I'm using this codebase as a practical playground to master Claude Code's full workflow — from scaffolding and refactoring, to debugging, architectural planning, automated code review, and integrating it deeply into a modern development loop.

The goal is twofold:

1. **Ship a clean, well-structured application** by iteratively improving an intentionally rough starter — fixing bugs, sharpening the UI, and refactoring toward maintainable, production-quality code.
2. **Build fluency with AI-assisted development**, learning how to delegate effectively, prompt precisely, and leverage an AI pair-programmer as a real force multiplier in day-to-day engineering.

## Tech Stack

- **React 19** — UI and state management
- **Vite** — dev server and build tooling
- **ESLint** (flat config) — linting with React Hooks and React Refresh plugins

## Getting Started

```bash
npm install
npm run dev
```

Then open your browser at `http://localhost:5173`.

## Scripts

| Command           | Description                          |
| ----------------- | ------------------------------------ |
| `npm run dev`     | Start the Vite dev server            |
| `npm run build`   | Build the app for production         |
| `npm run preview` | Preview the production build locally |
| `npm run lint`    | Run ESLint across the codebase       |
